# The Classical Excellence Roadmap
> A self‑directed Great Books curriculum focused on **Grammar → Logic → Rhetoric → Philosophy → History → Social Graces → Economics → Theology → Literature**.  
> Core document: **[`roadmap.md`](./roadmap.md)**

---

## Why this repo?

- To **learn deeply from primary sources** without getting lost in commentary.
- To **practice** (not just read): write, speak, and discuss each month.
- To keep everything in one place: reading plan, notes, and progress.

If you want a clear, minimal path through foundational texts (with built‑in practice), start here.
